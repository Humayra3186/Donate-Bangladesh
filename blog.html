Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects

How do you select an element from the DOM?

I usually select an element by using: document.getElementById('id')

What is event delegation in the context of the DOM, and why is it useful?

It refers the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements.

How do you manipulate an element's attributes and styles using DOM?

An element attribute can manipulate by using: getAttribute(), setAttribute(), removeAttribute() and can apply styles by using: element.style.propertyName = "value";
